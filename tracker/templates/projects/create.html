The file you select must be a '.xlsx' file with following columns: Name, Website, target, target_label, mailing_list.
Target labels are keywords, they must be ';' separated if numerours.
Mailing list are emails, they must be ';' separated if numerous as well.
Watchlist name will take the name of the xlsx file.
